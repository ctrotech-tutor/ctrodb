# ctrodb Examples

| Example | Description | Run |
|---|---|---|
| [CDN Todo](cdn/index.html) | Browser todo app via `<script>` tag | Open in browser |
| [Node.js](node/index.mjs) | CRUD, queries, FTS plugin in Node | `node examples/node/index.mjs` |

## Quick Links

- **CDN (script tag)** — Copy `cdn/index.html` and open in your browser. No build step needed.
- **Node.js** — Run `node examples/node/index.mjs` from the project root. Requires `npm run build` first.
- **React** — See [React Integration](src/react.ts) and the [react.test.tsx](../tests/unit/react.test.tsx) test file for API usage examples.
